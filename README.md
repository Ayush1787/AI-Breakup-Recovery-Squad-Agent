# AI-Breakup-Recovery-Squad-Agent
Multi-agent AI system for breakup recovery that delivers therapy-like support, closure writing, structured recovery plans, and honest advice using Groq-powered LLMs with a Flask web interface.
# 💔 AI Breakup Recovery Squad Agent

AI-powered emotional support system that helps users recover from breakups using multiple intelligent agents.

---

## 🌟 Features

* 🤗 Therapist Agent – Emotional support
* ✍️ Closure Agent – Express hidden feelings
* 📅 Recovery Planner – 7-day healing plan
* 💪 Brutal Honesty Agent – Real advice
* 🖼️ Image Support
* 🌐 Flask Web App
* 💬 CLI Mode

---

## ⚙️ Installation

### 1. Clone Repo

```
git clone https://github.com/Ayush1787/breakup-recovery-agent.git
cd breakup-recovery-agent
```

---

### 2. Install Dependencies

```
pip install flask agno groq python-dotenv markdown2
```

---

### 3. Setup API Key

Create `.env` file:

```
GROQ_API_KEY=your_api_key_here
```

---

## ▶️ Usage

### Run CLI

```
python test.py
```

---

### Run Web App

```
python app.py
```

Open:

```
http://127.0.0.1:5000/
```

---

## ⚠️ Security

* Never share API key
* Use `.env` file
* Add `.env` in `.gitignore`
